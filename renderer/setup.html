Linux tip:

For automatic project matching from window titles, install xdotool:

sudo apt install xdotool

On GNOME Wayland this works automatically. Without xdotool, the app still works — you just won't get auto-categorization from window context.

AI Setup Optional

HuminLoop uses Google's Gemini AI to auto-categorize your screenshots and enable natural language search. You can skip this and add it later in Settings.

Gemini API Key Recommended

Free tier available. Get a key from Google AI Studio in 30 seconds.

Vertex AI (Service Account)

For GCP users with existing billing. Uses a credentials.json file.

Skip for now

Use HuminLoop without AI. You can set this up later.

How to get your API key:
  1. Go to https://aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click Create API Key
  4. Copy the key and paste it below
Vertex AI setup:
  1. Go to https://console.cloud.google.com/
  2. Enable the Vertex AI API
  3. Create a Service Account under Credentials
  4. Download the JSON key file
  5. Save it as credentials.json in the HuminLoop folder
